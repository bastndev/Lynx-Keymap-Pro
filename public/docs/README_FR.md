> [!IMPORTANT]
>
> **L'`Accent Grave` = ( ` )** prend en charge toutes les dispositions de clavier 🌎
>
> ```ts
> exports.BACKTICK_EQUIVALENTS = [
>   { Backquote: "`" }, // Anglais, US
>   { Backquote: "º" }, // Espagnol (Espagne)
>   { Backquote: "'" }, // Français, Allemand
>   { Backquote: "²" }, // Français
>   { Backquote: "^" }, // Allemand, Suisse
>   { Backquote: "ё" }, // Russe
>   { Backquote: "~" }, // Portugais, Italien
>   { Backquote: "半角/全角" }, // Japonais
>   { Backquote: "|" }, // Espagnol (Amérique Latine)
>   { Backquote: "¬" }, // Royaume-Uni, Amérique Latine
>   { Backquote: "¨" }, // Allemand, Suédois
>   { Backquote: "´" }, // Portugais (Amérique Latine)
>   { Backquote: "₺" }, // Turc
>   { Backquote: "₤" }, // Italien
>   { Backquote: "" }, // Et PLUS ENCORE...
> ];
> ```

> [!NOTE]
>
> ### Raccourcis par Défaut Désactivés
>
> Cette extension désactive intentionnellement les raccourcis VS Code par défaut suivants afin d'éviter les conflits :
>
> - **`Ctrl+P` / `⌘+P`** — Ouverture Rapide (désactivé pour empêcher les conflits avec des raccourcis personnalisés)
>   - _Alternative_ : Utilisez `Shift+Alt+E` pour l'Ouverture Rapide
> - **`Ctrl+Enter` / `⌘+Enter`** — GitHub Copilot Generate (désactivé)
>   - _Raison_ : Conflits avec les flux de travail du terminal et des commits git
>
> Si vous avez besoin de ces raccourcis, vous pouvez les réactiver dans votre `keybindings.json` :
>
> ```json
> [
>   { "key": "ctrl+p", "command": "workbench.action.quickOpen" },
>   { "key": "ctrl+enter", "command": "github.copilot.generate" }
> ]
> ```
>
> ### Limitations Connues
>
> - **`Alt+CapsLock`** peut ne pas fonctionner sur certaines distributions Linux ou systèmes Windows où CapsLock est réassigné au niveau du système d'exploitation. Si ce raccourci ne fonctionne pas, vérifiez les paramètres du clavier de votre système ou utilisez des raccourcis alternatifs.

## Installation

### Méthode 1 — Ouverture Rapide

- <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> Linux `Ctrl+P`
- <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> macOS `⌘P`
- <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> Windows `Ctrl+P`

Collez la commande suivante et appuyez sur `Enter` :

```
ext install bastndev.lynx-keymap
```

### Méthode 2 — Vue des Extensions

1. Ouvrez les Extensions (`Ctrl+Shift+X` / `⌘+Shift+X`)
2. Recherchez **"Lynx Keymap Pro"**
3. Cliquez sur **Installer**
