> [!IMPORTANT]
>
> **`バッククォート` = ( ` )** はすべてのキーボードレイアウトをサポートしています 🌎
>
> ```ts
> exports.BACKTICK_EQUIVALENTS = [
>   { Backquote: "`" }, // 英語, 米国
>   { Backquote: "º" }, // スペイン語 (スペイン)
>   { Backquote: "'" }, // フランス語, ドイツ語
>   { Backquote: "²" }, // フランス語
>   { Backquote: "^" }, // ドイツ語, スイス
>   { Backquote: "ё" }, // ロシア語
>   { Backquote: "~" }, // ポルトガル語, イタリア語
>   { Backquote: "半角/全角" }, // 日本語
>   { Backquote: "|" }, // スペイン語 (ラテンアメリカ)
>   { Backquote: "¬" }, // 英国, ラテンアメリカ
>   { Backquote: "¨" }, // ドイツ語, スウェーデン語
>   { Backquote: "´" }, // ポルトガル語 (ラテンアメリカ)
>   { Backquote: "₺" }, // トルコ語
>   { Backquote: "₤" }, // イタリア語
>   { Backquote: "" }, // その他...
> ];
> ```

> [!NOTE]
>
> ### 無効化されたデフォルトショートカット
>
> この拡張機能は、競合を避けるために以下の VS Code デフォルトショートカットを意図的に無効化しています：
>
> - **`Ctrl+P` / `⌘+P`** — クイックオープン（カスタムショートカットとの競合を防ぐため無効化）
>   - _代替手段_：クイックオープンには `Shift+Alt+E` を使用してください
> - **`Ctrl+Enter` / `⌘+Enter`** — GitHub Copilot Generate（無効化）
>   - _理由_：ターミナルおよび git コミットワークフローとの競合
>
> これらのショートカットが必要な場合は、`keybindings.json` で再度有効化できます：
>
> ```json
> [
>   { "key": "ctrl+p", "command": "workbench.action.quickOpen" },
>   { "key": "ctrl+enter", "command": "github.copilot.generate" }
> ]
> ```
>
> ### 既知の制限事項
>
> - **`Alt+CapsLock`** は、一部の Linux ディストリビューションや、OS レベルで CapsLock が再マップされている Windows システムでは機能しない場合があります。このショートカットが機能しない場合は、システムのキーボード設定を確認するか、代替ショートカットを使用してください。

## インストール

### 方法 1 — クイックオープン

- <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> Linux `Ctrl+P`
- <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> macOS `⌘P`
- <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> Windows `Ctrl+P`

次のコマンドを貼り付けて `Enter` を押します:

```
ext install bastndev.lynx-keymap
```

### 方法 2 — 拡張機能ビュー

1. 拡張機能を開く (`Ctrl+Shift+X` / `⌘+Shift+X`)
2. **「Lynx Keymap Pro」**を検索
3. **インストール**をクリック
