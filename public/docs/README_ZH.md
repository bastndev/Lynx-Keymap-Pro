> [!IMPORTANT]
>
> **`反引号 (Backquote)` = ( ` )** 支持所有键盘布局 🌎
>
> ```ts
> exports.BACKTICK_EQUIVALENTS = [
>   { Backquote: "`" }, // 英语 (美国)
>   { Backquote: "º" }, // 西班牙语 (西班牙)
>   { Backquote: "'" }, // 法语, 德语
>   { Backquote: "²" }, // 法语
>   { Backquote: "^" }, // 德语, 瑞士
>   { Backquote: "ё" }, // 俄语
>   { Backquote: "~" }, // 葡萄牙语, 意大利语
>   { Backquote: "半角/全角" }, // 日语
>   { Backquote: "|" }, // 西班牙语 (拉丁美洲)
>   { Backquote: "¬" }, // 英国, 拉丁美洲
>   { Backquote: "¨" }, // 德语, 瑞典语
>   { Backquote: "´" }, // 葡萄牙语 (拉丁美洲)
>   { Backquote: "₺" }, // 土耳其语
>   { Backquote: "₤" }, // 意大利语
>   { Backquote: "" }, // 还有更多...
> ];
> ```

> [!NOTE]
>
> ### 禁用的默认快捷键
>
> 此扩展故意禁用以下默认的 VS Code 快捷键以避免冲突：
>
> - **`Ctrl+P` / `⌘+P`** — 快速打开 (禁用以防止与自定义快捷键冲突)
>   - _替代方案_: 使用 `Shift+Alt+E` 进行快速打开
> - **`Ctrl+Enter` / `⌘+Enter`** — GitHub Copilot 生成 (禁用)
>   - _原因_: 与终端和 git 提交的工作流发生冲突
>
> 如果您需要这些快捷键，可以在 `keybindings.json` 中重新启用它们：
>
> ```json
> [
>   { "key": "ctrl+p", "command": "workbench.action.quickOpen" },
>   { "key": "ctrl+enter", "command": "github.copilot.generate" }
> ]
> ```
>
> ### 已知限制
>
> - **`Alt+CapsLock`** 可能在某些 Linux 发行版或 Windows 系统上无法工作，因为 CapsLock 键在操作系统层面被重新映射了。如果此快捷键不起作用，请检查您的系统键盘设置或使用替代快捷键。

## 安装说明

### 方法 1 — 快速打开

- <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> Linux `Ctrl+P`
- <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> macOS `⌘P`
- <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> Windows `Ctrl+P`

粘贴以下命令并按下 `Enter` 键：

```
ext install bastndev.lynx-keymap
```

### 方法 2 — 扩展视图

1. 打开扩展面板 (`Ctrl+Shift+X` / `⌘+Shift+X`)
