> [!IMPORTANT]
>
> **`백틱` = ( ` )**은 모든 키보드 레이아웃을 지원합니다 🌎
>
> ```ts
> exports.BACKTICK_EQUIVALENTS = [
>   { Backquote: "`" }, // 영어, 미국
>   { Backquote: "º" }, // 스페인어 (스페인)
>   { Backquote: "'" }, // 프랑스어, 독일어
>   { Backquote: "²" }, // 프랑스어
>   { Backquote: "^" }, // 독일어, 스위스
>   { Backquote: "ё" }, // 러시아어
>   { Backquote: "~" }, // 포르투갈어, 이탈리아어
>   { Backquote: "半角/全角" }, // 일본어
>   { Backquote: "|" }, // 스페인어 (라틴 아메리카)
>   { Backquote: "¬" }, // 영국, 라틴 아메리카
>   { Backquote: "¨" }, // 독일어, 스웨덴어
>   { Backquote: "´" }, // 포르투갈어 (라틴 아메리카)
>   { Backquote: "₺" }, // 터키어
>   { Backquote: "₤" }, // 이탈리아어
>   { Backquote: "" }, // 기타...
> ];
> ```
