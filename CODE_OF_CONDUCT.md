# Code of Conduct

## Our Commitment

Lynx Keymap pro is committed to a welcoming, respectful environment for everyone — regardless of skill level, keyboard layout, or editor preference.

## Standards

**We encourage:** respectful interactions, constructive feedback, patience with newcomers, knowledge sharing about pro layouts, and focusing on improving the experience for all users.

**We do not tolerate:** harassment, personal attacks, discriminatory comments, publishing private information, keyboard elitism, or any conduct that creates a hostile environment.

## Contributions

We value all forms of contribution: bug reports, keybinding improvements, new editor support, documentation, screenshots, and community help.

See [CONTRIBUTING.md](https://github.com/bastndev/Lynx-Keymap-pro/blob/main/CONTRIBUTING.md) for detailed guidelines.

## Enforcement

Report violations to **bastndev@gohit.xyz** using the subject format:

```
[Lynx Keymap Pro] — Brief description
```

All reports are handled confidentially. Depending on severity:

1. **First violation** — Private warning and clarification
2. **Repeated issues** — Temporary restriction from project spaces
3. **Serious violations** — Permanent removal from the community

## Scope

This Code of Conduct applies to all project spaces: GitHub issues, pull requests, discussions, and any public forum where this project is represented.
